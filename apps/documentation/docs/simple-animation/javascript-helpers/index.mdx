# JavaScript Helpers

The JavaScript helpers are used to enhance certain animations and/or provide additional functionality.

These helpers are **not** required to use the CSS animations.

## Getting Started

The use of a JavaScript helper will depend on which animations you are using. Not all animations require a helper.

The helper list below will indicate which animations require a helper and each will provide details on how to use them in your project.
